# 销售开单系统

一个纯前端的销售开单管理工具，单 HTML 文件即可运行，数据存储在浏览器本地。

## 功能

- **🖊️ 开单** — 创建销售订单，支持客户搜索、货品选择、多费用项、自动金额计算（中文大写）、打印票据
- **👥 客户管理** — 新增/编辑/删除客户信息（编码、名称、联系方式、地址等）
- **📦 货品管理** — 管理货品编号、名称、规格、单价，支持两种计价模式
- **📋 订单记录** — 查看/搜索/编辑/删除历史订单，导出 CSV 报表
- **💾 数据导入导出** — JSON 格式备份与恢复
- **🖨️ 打印票据** — 自动生成格式化的销售票据打印页
- **⚙️ 系统设置** — 自定义公司名称、编码前缀、默认备注、费用项等

## 使用方式

直接在浏览器中打开 `https://sharksame-owgosu.github.io/XIAOSHOUHTMLTOOL/kaidan-order-oss-v1.0.html` 即可使用，无需服务器。

数据保存在浏览器 localStorage 中，建议定期使用"导出数据"功能备份。
